Опис проєкту

Ця лабораторна робота присвячена вивченню принципів створення графічного інтерфейсу користувача (GUI) у мові Java за допомогою бібліотеки Swing.

У ході виконання роботи було реалізовано програму для розрахунку кількості теплоти, що утворюється при згорянні речовини, а також реалізовано можливість оберненого розрахунку, коли за будь-якими двома відомими величинами автоматично визначається третя.

Загальна реалізація

Проєкт реалізовано з використанням принципів об'єктно-орієнтованого програмування.

Структура програми складається з трьох окремих класів:

Main.java

Головний клас програми.
Містить метод main(), який запускає графічне вікно.

HeatCalculator.java

Клас математичних обчислень.
Містить методи:

calculateQ() — обчислення кількості теплоти
calculateM() — обчислення маси речовини
calculateq() — обчислення питомої теплоти згоряння
CalculatorFrame.java

Клас графічного інтерфейсу.
Відповідає за:

створення вікна програми
розміщення елементів керування
зчитування введених даних
перевірку правильності вводу
виконання обчислень після натискання кнопки
Реалізовані можливості програми
Основний функціонал

Програма дозволяє:

вводити значення маси m

вводити кількість теплоти Q
натискати кнопку «Обчислити»
автоматично знаходити невідому величину

Використані компоненти Swing

У програмі використані такі компоненти:

JFrame — головне вікно
JLabel — текстові підписи
JTextField — поля вводу
JButton — кнопка запуску
JOptionPane — повідомлення про помилки
Як запустити локально

Відкрийте термінал у папці з файлами проєкту.
