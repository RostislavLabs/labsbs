Лабораторна робота 9. Swing. Варіант 4.

Тема: графічний інтерфейс користувача на Java Swing.
Завдання: за законом Архімеда розрахувати умову плавання тіла масою m та об'ємом V,
зануреного у рідину з густиною ρ.

Структура:
- src/Main.java — запуск програми;
- src/ArchimedesFrame.java — графічне вікно JFrame, поля JTextField, кнопки JButton;
- src/ArchimedesCalculator.java — усі фізичні обчислення;
- src/InputValidator.java — перевірка введення через try-catch;
- src/PhysicsConstants.java — фізична стала g = 9.81;
- bin — папка для .class файлів;
- lib — папка для бібліотек;
- .vscode/settings.json — налаштування VS Code, щоб .class створювались у bin.

Запуск у терміналі з папки Lab_9:
javac -encoding UTF-8 -d bin src\*.java
java -cp bin Main

Як користуватися:
1. Для прямої задачі введіть m, V, ρ і натисніть "Обчислити".
2. Для оберненої задачі введіть тільки два значення з трьох: m, V, ρ.
   Третє поле автоматично блокується і після натискання кнопки обчислюється для рівноваги плавання.
3. Якщо замість числа ввести літери, програма покаже повідомлення про помилку введення.


Для запуску коду
cd Lab_9
javac -encoding UTF-8 -d bin src\*.java
java -cp bin Main
