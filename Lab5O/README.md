Опис проєкту

Ця лабораторна робота присвячена вивченню принципів створення та використання методів у мові Java. Програма демонструє побудову окремих методів для виконання математичних обчислень, роботи з масивами та пошуку значень. Також показано багатофайлову структуру проєкту, де кожне завдання винесене в окремий клас, що робить код більш зрозумілим, структурованим і зручним для супроводу.

Вимоги до середовища

Для компіляції та запуску програми необхідний встановлений JDK (Java Development Kit).

Загальна реалізація

Код побудований за принципами об'єктно-орієнтованого програмування. Проєкт складається з головного класу Main, який є точкою входу в програму, та чотирьох окремих класів:

Triangle – обчислення площі прямокутного трикутника;
MinNumber – пошук найменшого з трьох чисел;
ArrayPrinter – виведення масиву на екран;
MaxNumber – пошук найбільшого елемента масиву.

Головний клас послідовно викликає методи кожного класу та виводить результати роботи програми.

Реалізовані завдання
Завдання 1 (Обчислення площі прямокутного трикутника)

Створено метод area(double a, double b), який приймає два параметри: катети трикутника. Метод обчислює площу за формулою:

S = (a * b) / 2

та повертає результат типу double.

Завдання 2 (Пошук найменшого числа)

Створено метод findMin(int a, int b, int c), який приймає три цілі числа, знаходить серед них найменше та повертає результат.

Завдання 3 (Виведення масиву)

Створено метод printArray(int[] arr), який приймає масив цілих чисел і виводить усі його елементи в один рядок через пробіл.

Завдання 4 (Пошук найбільшого числа з масиву)

Створено метод findMax(int[] arr), який приймає масив чисел, знаходить найбільший елемент та повертає його значення.

Як запустити локально
Відкрити термінал та перейти в папку з файлами лабораторної роботи.
Переконатися, що встановлений JDK:
javac -version
Скомпілювати всі файли програми:
javac *.java
Запустити головний клас програми:
java Main
