🧾 Опис проєкту

Ця лабораторна робота присвячена вивченню принципів роботи з потоками введення та виведення в мові Java, а також роботі з файлами.
Програма демонструє створення текстового файлу, запис у нього даних та подальше зчитування інформації.

У ході виконання роботи використано символьні потоки введення/виведення (FileWriter та FileReader), а також клас File для перевірки існування файлу.

Основною функціональністю програми є генерація послідовності випадкових чисел у заданому діапазоні, запис їх у файл та виведення вмісту файлу на консоль.

⚙️ Вимоги до середовища

Для компіляції та успішного запуску програми необхідно:

Встановлений JDK (Java Development Kit)
Будь-яке середовище розробки або термінал (PowerShell / CMD / Terminal)
🏗️ Загальна реалізація

Код реалізовано з використанням принципів об'єктно-орієнтованого програмування та розбитий на декілька класів, кожен з яких відповідає за окрему функціональність:

Main — головний клас, точка входу в програму
FileManager — відповідає за роботу з файлами (створення, запис, читання)
RandomGenerator — генерує випадкові числа у заданому діапазоні
Validator — перевіряє коректність аргументів командного рядка

Такий підхід забезпечує модульність, зручність читання та повторне використання коду.

🧩 Реалізоване завдання
🔹 Генерація та запис даних у файл
Програма отримує параметри командного рядка:
ім’я файлу
початок діапазону (b1)
кінець діапазону (b2)
кількість чисел
Виконується перевірка правильності введених даних
Генерується масив випадкових чисел у заданому діапазоні
Дані записуються у текстовий файл за допомогою FileWriter
🔹 Перевірка існування файлу
Перед записом перевіряється, чи існує файл
Якщо файл відсутній — він автоматично створюється
Використано клас File
🔹 Зчитування даних з файлу
Після запису програма відкриває файл
Дані зчитуються посимвольно за допомогою FileReader
Вміст файлу виводиться у консоль

У результаті виконання програми:

Вміст файлу виводиться на консоль
