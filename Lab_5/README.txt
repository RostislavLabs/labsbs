Lab_5 — багатофайловий Java-проєкт

Структура проєкту:
Lab_5
├── src
│   ├── Main.java
│   ├── Task1_Rhombus.java
│   ├── Task2_MinOfThree.java
│   ├── Task3_ArrayPrint.java
│   └── Task4_ArrayMax.java
├── bin
└── lib

Запуск через VS Code Terminal / PowerShell:

cd Lab_5
javac -encoding UTF-8 -d bin src\*.java
java -cp bin Main

Що виконує програма:
1. Обчислює площу ромба за діагоналями.
2. Знаходить найменше з трьох чисел.
3. Виводить масив.
4. Знаходить найбільше число в масиві.
