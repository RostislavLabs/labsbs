Опис проєкту

Ця лабораторна робота присвячена вивченню циклічних конструкцій (for, while), обробці введення даних з клавіатури за допомогою класу Scanner та використанню вбудованих математичних методів класу Math. Програма демонструє різні способи організації повторюваних обчислень, умовних операторів та базових алгоритмів.

Вимоги до середовища

Для компіляції та запуску програми необхідно:

встановлений JDK (Java Development Kit)
середовище виконання Java (JRE)
Загальна реалізація

Програма реалізована в одному класі Lab3, який містить послідовні блоки виконання у методі main.
Кожне завдання виконується по черзі.
Використано:

умовні оператори (if-else)
цикли (for, while)
математичні функції (Math)
введення даних (Scanner) з перевіркою коректності
Реалізовані завдання
Завдання 1 (Обчислення виразу)

Обчислюється значення математичного виразу з використанням степеня та квадратного кореня. Продемонстровано використання методів Math.pow() та Math.sqrt().

Завдання 2 (Розгалуження)

Реалізовано обчислення функції з використанням умовного оператора if-else.
Значення функції залежить від введеного користувачем числа.

✔ Додатково реалізовано:

перевірку введення (захист від введення символів замість числа)
Завдання 3 (Цикл for)

Обчислюється сума чисел від 1 до 10 за допомогою циклу for.

Завдання 4 (Цикл while)

Обчислюється факторіал числа 5 за допомогою циклу while.

Завдання 5 (Табулювання функції)

Обчислюється функція:

[
f(x) = 3 \sin(\sqrt{3} + 0.35x) - 3.8
]

на проміжку від -2 до 2 з кроком 0.2.

✔ Реалізовано:

табулювання функції двома способами:
за допомогою циклу for
за допомогою циклу while
форматований вивід результатів
