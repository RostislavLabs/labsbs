Опис проєкту

Ця лабораторна робота присвячена вивченню принципів роботи з класами в мові Java. Програма демонструє створення власних класів, використання полів і методів, роботу з конструкторами, перевантаження методів, спадкування та абстрактні класи. Також реалізовано індивідуальне завдання згідно варіанту №12.

Вимоги до середовища

Для компіляції та успішного запуску коду необхідний встановлений JDK (Java Development Kit).

Загальна реалізація

Код структуровано за принципами об'єктно-орієнтованого програмування. Проєкт складається з головного класу Main, який слугує точкою входу, та окремих класів для кожного завдання: Person, Phone, Car, Sedan, Truck, Product, SmartProduct.

Головний клас послідовно створює об’єкти класів та викликає їх методи, що робить програму модульною, зрозумілою та зручною для читання.

Реалізовані завдання
Завдання 1 (Клас Person)

Створено клас Person, який містить поля fullName та age.

Реалізовано два конструктори:

без параметрів;
з параметрами fullName і age.

Додано методи:

move() – виводить повідомлення про рух людини;
talk() – виводить повідомлення про розмову людини.

Створено два об’єкти класу з різними способами ініціалізації.

Завдання 2 (Клас Phone)

Створено клас Phone, який містить поля:

номер телефону;
модель;
вага.

Реалізовано три конструктори:

без параметрів;
з двома параметрами;
з трьома параметрами.

Додано методи:

receiveCall() – прийом дзвінка;
перевантажений receiveCall() з номером телефону;
getNumber() – повертає номер телефону;
sendMessage() – приймає довільну кількість номерів і виводить їх на екран.
Завдання 3 (Ієрархія класів Car)

Створено абстрактний клас Car, який містить поля:

модель;
колір;
максимальна швидкість.

Реалізовано методи:

gas() – звичайний метод;
brake() – абстрактний метод.

Створено похідні класи:

Sedan;
Truck.

Для кожного класу перевизначено метод гальмування.

Завдання 4 (Індивідуальний варіант №12)

Створено клас Product, який описує товар.

Дані класу:

найменування товару;
ціна товару;
термін гарантії.

Реалізовано методи:

виведення даних товару;
зниження ціни на 50%.
Завдання 5 (Клас-нащадок)

На основі класу Product створено клас SmartProduct, який розширює можливості базового класу.

Додано нові поля:

бренд;
наявність бездротового режиму.

Перевизначено метод виведення інформації про товар.

Як запустити локально

Відкрийте термінал та перейдіть у папку з файлами лабораторної роботи.

Переконайтеся, що у вас встановлено JDK (можна перевірити командою javac -version).

Скомпілюйте одразу всі класи лабораторної роботи:

javac *.java

Запустіть головний файл програми:

java Main
