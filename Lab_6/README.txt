Лабораторна робота №6. Варіант 4.

Структура проєкту:
- src - Java-файли програми;
- bin - папка для скомпільованих .class файлів;
- lib - папка для бібліотек, якщо вони знадобляться.

Запуск з терміналу PowerShell або CMD:
1) Перейдіть у папку Lab_6:
   cd шлях_до_папки\Lab_6

2) Скомпілюйте програму:
   javac -encoding UTF-8 -d bin src\*.java

3) Запустіть програму:
   java -cp bin Main

У програмі реалізовано:
1. Завдання Person.
2. Завдання Phone.
3. Ієрархію Car / Sedan / Truck.
4. Індивідуальне завдання варіанту 4: клас прямокутного трикутника.
5. Клас-спадкоємець ExtendedRightTriangle.
6. Перевірку введення: якщо замість числа ввести букву, програма просить ввести число ще раз.


Ввести в терміналі для запуску:
cd Lab_6
